# fundamental-in-ai-ml-project
Introduction - 
The Weather Information System is a Python-based application designed to provide real-time weather updates for any city entered by the user. It is a simple yet effective project for beginners to understand how real-time data can be fetched, processed, and displayed in a user-friendly format.
Objective - 
The main objective of this project is to build a system that can retrieve and display accurate weather information such as temperature, wind speed, and time.
Features - 
Retrieves real-time weather data for any city
Displays temperature and “feels like” temperature and wind speed. Handles invalid inputs and network errors gracefully. Allows repeated queries without restarting the program
Technologies Used - 
Python – Core programming language
Imported Library
Installation and Setup
Install Python (version 3.x recommended) on your system.
Install the required library by running the command:
pip install requests
Save the program file and run it using any Python IDE or terminal.
Usage - 
After running the program, the user is prompted to enter a city name. Once the input is provided, the program retrieves weather data. The information is then displayed in a structured and readable format. The program continues to run until the user types “exit,” allowing multiple searches in one session.
Error Handling
The program includes basic error handling to improve reliability. If the user enters an invalid city name, the system displays an appropriate error message. It also handles network-related issues to prevent the program from crashing unexpectedly.
Conclusion - 
This Weather Information System is a practical project that introduces beginners to real-world programming concepts. It is easy to understand, user-friendly, and can be enhanced further by adding features, weather forecasts, or data storage capabilities.
